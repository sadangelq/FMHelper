# FMHelper

Мод для Minecraft с системными сообщениями, кнопкой телепортации в хаб и автоматическим обновлением.

## Возможности

- ✦ Системные сообщения с разными типами (INFO, SUCCESS, WARNING, ERROR)
- ⌂ Кнопка быстрой телепортации в хаб
- ⟳ Автоматическое обновление мода прямо в игре
- ⚙️ Полная настройка всех функций в меню конфигурации

## Требования

- Minecraft 1.21.8
- Fabric Loader 0.18.4+
- Java 21+
- Fabric API

## Установка

1. Скачайте последнюю версию мода с [Releases](https://github.com/sadangelq/FMHelper/releases)
2. Поместите JAR файл в папку `mods`
3. Запустите Minecraft с профилем Fabric
4. В главном меню нажмите кнопку обновлений (⟳) для проверки доступных версий

## Использование

### В главном меню

- **⟳** - Проверить обновления мода
- **⌂** - Телепортироваться в хаб

### Конфигурация

Все настройки мода доступны в меню "Mods" → "Конфиг FMHelper"

### Системные сообщения

Мод автоматически показывает:
- Приветствие при входе на сервер
- Сообщения о низком здоровье
- Сообщения о низком голоде
- Сообщения о смерти
- Уведомления о входе в новый биом

## Автообновление

Мод автоматически проверяет наличие новых версий при входе в игру и уведомляет вас через системные сообщения.

### Как обновиться

1. Откройте главное меню
2. Нажмите кнопку обновлений (⟳)
3. Нажмите "Загрузить обновление"
4. Перезагрузите игру

Старая версия мода будет сохранена как `FMHelper-X.X.jar.bak` для безопасности.
